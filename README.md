Script for quantitative trading. First, script to predict 21-day returns. There is a short version that only uses technical indicators and a full version that adds additional data. The short version uses only half of the data for hyperparameter tuning, this is very efficient and OOB metrics show no difference to the full training. Also the short version with only technical indicators usually shows lower RMSE, higher R2 and faster training with lower memory usage than the full data.
Also are scripts for some backtesting and trading on Interactive Brokers (IB). This requires an IB account and use of the Trader Workstation in the background to open the API connection.
Return so far on 2026 has been good, well above the S&P 500 index (19.March 2026). However there is no guarantee, this is not investing advice, scripts are for education only. Use at your own risk.
